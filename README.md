# Internship-portfolio-todo
Personal Portfolio integrated with a Functional To-Do List | Built with HTML, CSS, JavaScript 
This project is a personal portfolio integrated with a functional To-Do List.  
Built with HTML, CSS, and JavaScript, it showcases my skills and allows task management in a simple and interactive way.

**Features:**  
- Personal portfolio with sections: About, Projects, Skills  
- Functional To-Do List: Add, Complete, Clear tasks  
- Tasks saved in localStorage
**How to Run:**  
1. Clone or download this repository  
2. Open `index.html` in any web browser  
3. Add tasks in the To-Do List and manage them interactively
